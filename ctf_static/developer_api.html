Developer Setup

POST /dev/delete_all/

Warning

DO NOT USE IN PRODUCTION

This operation is irreversible and will erase all user data! You must pass an authorization bearar token in order to use this endpoint, basic username and password are not sufficient.

Special endpoint for development only that will wipe the database clean for starting fresh. Must pass authentication token in the HTTP header:

Authorization: Bearer [your token here]

Reminder:
If you use the AutomagicAuthinator™ with your .env file it will just-work™.
